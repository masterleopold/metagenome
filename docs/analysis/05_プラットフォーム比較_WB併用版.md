#### 未知病原体検出シナリオ例

**シナリオ1: 新規ウイルス発見（例: 新型porcine coronavirus）**

```yaml
MinION/PromethION アプローチ:
  1. Kraken2分類: "unclassified reads" 検出（5-10%）
  2. metaFlye assembly: 30 kb完全ゲノム取得（単一コンティグ）
  3. BLAST解析: 既知coronavirusと75%相同性
  4. Annotation: ORF1ab, spike, envelope遺伝子同定
  5. 系統解析: Betacoronavirus属、新規種と判定
  6. 結論: 72時間以内に新規ウイルス同定 ✅

  利点:
    ✅ 完全ゲノム配列取得
    ✅ リアルタイム検出（sequencing中）
    ✅ 即座にPCR設計可能
    ✅ PMDA報告用データ完備

MiSeq アプローチ:
  1. Kraken2分類: "unclassified reads" 検出
  2. SPAdes assembly: 300 bp断片×100コンティグ
  3. BLAST解析: 部分的相同性のみ
  4. Gap filling: 困難（リード長不足）
  5. 結論: 完全ゲノム取得に追加実験必要 ⚠️

  課題:
    ⚠️ 断片的配列のみ
    ⚠️ ギャップ多数
    ⚠️ Sanger sequencing追加必要
    ❌ 即座の系統解析困難
```

**シナリオ2: 既知ウイルスの新規変異株（例: PERV組換え体）**

```yaml
MinION/PromethION 検出能力:
  - PERV-A/B組換え: 完全検出 ✅
  - Env gene変異: 単一リードで検出
  - LTR重複: 構造解析可能
  - 起源特定: 豚系統同定可能

  例: PERV-A/B組換え体検出
    → 30 kbリードで完全PERV provirus取得
    → Env遺伝子でPERV-A/B組換え検出
    → LTR解析で活性型判定
    → Western Blot確認で感染性評価

MiSeq 検出能力:
  - 点変異: 検出可能 ✅
  - 組換え: 検出困難 ❌（300 bp限界）
  - 構造変異: 検出不可 ❌
  - 完全provirus: 取得困難 ⚠️

  制限:
    ❌ 組換え検出にはlong-range PCR追加必要
    ⚠️ 構造変異見逃しリスク
```

**シナリオ3: 細菌の抗生物質耐性遺伝子（例: 新規プラスミド）**

```yaml

  - プラスミド完全配列: 取得可能（50-100 kb）
  - 耐性遺伝子クラスター: 単一リードで検出
  - 可動性因子: トランスポゾン・インテグロン構造解析

  - 耐性遺伝子: 検出可能（既知のみ）
  - プラスミド: 断片的（完全構造不明）
  - 可動性因子: 推定困難
  - 新規遺伝子: 見逃しリスク ⚠️
```

### 3.2 Western Blot判定基準

| NGS結果                  | Western Blot結果    | 最終判定            | 頻度（推定） | アクション                       |
| ---------------------- | ----------------- | --------------- | ------ | --------------------------- |
| Negative               | Negative          | **ACCEPT** ✅    | 10-20% | 優先ドナー、即時使用可能                |
| Negative               | Positive          | REJECT ❌        | \<1%   | 低レベルPERV（NGS検出限界以下）         |
| Positive               | Negative          | **ACCEPT** ⚠️   | 20-40% | Genomic PERV（非感染性）、強化モニタリング |
| Positive（50-100 gc/mL） | Weak/Inconclusive | Co-culture → 判定 | 5-10%  | ボーダーライン症例                   |
| Positive               | Positive          | REJECT ❌        | 30-50% | Functional PERV（感染性）        |

**Key Insight**:
- NGS陽性 + WB陰性 = Genomic PERV（許容可能、ドナープール拡大）
- NGS陰性 + WB陽性 = 稀だが要注意（NGS感度不足）
