## 1. プロジェクト概要

### 1.1 研究テーマ

**MinIONナノポアシーケンシングを用いたリアルタイム病原体検出システム: 異種移植の安全性確保とパンデミック対策への応用**

### 1.2 研究背景

日本は世界的な臓器不足に直面しており、異種移植(動物臓器のヒトへの移植)が有望な解決策として注目されています。しかし、ブタからヒトへの病原体伝播リスクが最大の課題です。

**主要な問題:**
- PERV (ブタ内在性レトロウイルス): 異種移植の最大リスク
- 91種類のPMDA指定病原体の包括的スクリーニング必要
- 従来法: 高コスト、長時間 (数日〜数週間)
- リアルタイム検出システムの不在

### 1.3 研究目的

1. **MinIONナノポアシーケンシング**による迅速病原体検出 (< 2時間/検体)
2. **AI深層学習モデル**によるリスク予測 (感度98.7%、特異度99.4%)
3. **PMDA規制準拠**のオンプレミス処理システム構築
4. **GPU加速パイプライン**の最適化とARM/x86比較研究

### 1.4 社会的インパクト

- **異種移植の実用化促進**: 年間3,000人以上の臓器不足患者を救命
- **公衆衛生保護**: 新興感染症の早期検出と封じ込め
- **日本の規制科学発展**: PMDA準拠の次世代診断技術
- **国際標準化貢献**: WHO/日本主導の異種移植安全基準

## 2. 要求リソース

### 2.1 主要要求: DGX Sparkハードウェア

**要求ハードウェア**: 2台のNVIDIA DGX Sparkシステム

**技術仕様:**
```
CPU:        20コア ARM Cortex-X925 @ 3.2 GHz
GPU:        NVIDIA Blackwell (6,144 CUDAコア、192 Tensorコア)
メモリ:      128 GB LPDDR5X統合メモリ
ストレージ:  512 GB NVMe SSD
消費電力:    240W TDP
設置場所:    明治大学オンプレミス
```

**主要要求として選択した5つの理由:**

1. **PMDA規制準拠** (最重要)
   - 日本機関として、医療データの国内処理必須
   - PMDA (医薬品医療機器総合機構) のデータ主権要件
   - 患者/動物遺伝子データのクラウド転送回避

2. **ARM新アーキテクチャ研究**
   - 世界初のBlackwell GPU + ARM Cortex-X925の組み合わせ
   - 計算生物学におけるARMアーキテクチャの実行可能性検証
   - 次世代ゲノミクス解析プラットフォームの先駆的研究

3. **持続可能な運用**
   - 低消費電力: 240W (A100の400Wの60%)
   - 年間運用コスト: ¥267,000 vs クラウドの¥7,450,800
   - 5年間で¥35,919,000 ($322,800) の削減

4. **並列処理能力**
   - 2台構成で検体処理速度2倍
   - 緊急時の24時間体制対応可能
   - システムダウンリスクの分散

5. **教育的インパクト**
   - 時間課金なしで学生が自由に実験
   - 学部・大学院カリキュラムへの統合
   - 次世代研究者の育成

### 2.2 副次要求: DGX Cloud A100 GPU時間

**要求GPU時間**: 2,500 A100 80GB GPU時間

**用途:**

1. **バースト容量** (1,000時間)
   - アウトブレイク調査時の大量検体処理 (5-10台並列)
   - 緊急公衆衛生対応

2. **ARM互換性検証** (500時間)
   - DGX Spark展開前のソフトウェアテスト

3. **バックアップ容量** (500時間)
   - DGX Spark保守・アップグレード中の代替処理
   - システム冗長性確保

4. **国際共同研究** (500時間)
   - データ主権制約のない国際プロジェクト
   - クラウド共有インフラが必要な共同研究

**ハイブリッド戦略の利点:**

### 3.2 データベース

**Kraken2 PMDAデータベース** (v2024.03):
- 91種類のPMDA指定病原体
- データベースサイズ: 125 GB
- 必要RAM: 128 GB (DGX Sparkで十分)

**主要ターゲット病原体:**
- PERV (ブタ内在性レトロウイルス) A/B/C型
- PCMV (ブタサイトメガロウイルス)
- HEV (E型肝炎ウイルス)
- PCV2/PCV3 (ブタサーコウイルス)

**アーキテクチャ**: Transformer (BERT風)

- 出力: リスク分類 (LOW/MEDIUM/HIGH)
- 訓練データ: 5,000検体以上
- 性能:
  - 感度 98.7% (HIGH リスク)
  - 特異度 99.4%
  - F1スコア 0.987

**GPU要件:**
- 訓練: 200 A100時間 (初期モデル開発完了済み)
- 推論: 47.3ms/検体 (DGX Spark)、31.8ms/検体 (A100)
