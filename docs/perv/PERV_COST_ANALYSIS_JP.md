## 臨床試験モニタリングコスト

### 10患者 × 5年間のモニタリング

#### 検査頻度

```
Year 1: 週次（1-4週） + 月次（2-12月） = 4 + 11 = 15回
Year 2: 月次 × 12 = 12回
Year 3: 四半期 × 4 = 4回
Year 4: 四半期 × 4 = 4回
Year 5: 半年 × 2 = 2回
------------------------------------------------------
合計: 37回/患者
10患者: 370サンプル
```

#### 方法別コスト

**方法A：個別MinION検査**
```
コスト: ¥162,000 × 370 = ¥59,940,000
期間: 370サンプル ÷ 24/日 = 15.4日
問題: 高コスト
```

**方法B：個別ELISA**
```
コスト: ¥35,000 × 370 = ¥12,950,000
期間: 370サンプル ÷ 96/日 = 3.9日
利点: 低コスト、高throughput
```

**方法C：プール検査（超推奨）⭐**

```
原理:
- 10患者の血漿を1つにプール → 1検査
- 陰性なら全員陰性（10倍効率化）
- 陽性なら10患者を個別再検査

コスト計算（MinION使用）:
プール検査: 37回 × ¥162,000 = ¥5,994,000
陽性時個別検査（5%と仮定）:
  - 1.85回陽性 × 10患者 × ¥162,000 = ¥2,997,000
合計: ¥8,991,000

節約: ¥59,940,000 - ¥8,991,000 = ¥50,949,000（85%削減）
```

**方法D：プール検査（ELISA）**

```
コスト計算:
プール検査: 37回 × ¥35,000 = ¥1,295,000
陽性時個別検査: 1.85回 × 10患者 × ¥35,000 = ¥647,500
Western確認: 1.85回 × ¥55,000 = ¥101,750
合計: ¥2,044,250

節約: ¥12,950,000 - ¥2,044,250 = ¥10,905,750（84%削減）
```
