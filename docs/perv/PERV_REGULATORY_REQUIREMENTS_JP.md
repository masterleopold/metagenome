# PERV検証：規制要件とコンプライアンス

**対象**: FDA/PMDA異種移植規制への準拠

---- 

## エグゼクティブサマリー

### 規制当局の要件

**FDA（米国）と PMDA（日本）の共通要件**:
	必須検証:
	✅ 全ゲノムシーケンス（CRISPR編集確認）
	✅ 共培養アッセイ（感染性ウイルス不在）
	✅ レシピエント長期モニタリング
	
	推奨検証（必須ではない）:
	⚪ Western BlotまたはELISA（タンパク質レベル）
	⚪ ddPCR（コピー数定量）
	
	明示的に必須ではない:
	❌ Flow cytometry
	❌ 免疫組織化学（IHC）
	❌ 質量分析

## FDA要件（米国）

### 主要ガイダンス文書

**"Source Animal, Product, Preclinical, and Clinical Issues Concerning the Use of Xenotransplantation Products in Humans"**
- 発行: 2016年（最新更新2024年）
- 対象: 異種移植製品の全段階

### Pre-Clinical Requirements（前臨床）

#### 1. Source Animal Characterization（ドナー動物特性評価）

	必須項目:
	┌────────────────────────────────────────────────┐
	│ A. 遺伝的特性評価                              │
	│    ✅ 全ゲノムシーケンス（30-50×カバレッジ）   │
	│    ✅ CRISPR編集部位の確認（on-target）        │
	│    ✅ Off-target変異解析                       │
	│    ✅ 多世代安定性（F0, F1, F2）               │
	│                                                │
	│ B. PERV検証                                    │
	│    ✅ Nucleic acid testing (NAT)               │
	│       - PCRまたはddPCRでPERV DNA検出           │
	│       - RT-PCRでPERV RNA検出                   │
	│    ✅ 感染性ウイルスアッセイ                   │
	│       - 共培養法（ヒト細胞、8週間）            │
	│       - RT活性測定                             │
	│    ⚪ 血清学的検査（推奨）                     │
	│       - Western BlotまたはELISA                │
	│       - ≥2抗原で判定                           │
	│                                                │
	│ C. 他の病原体スクリーニング                   │
	│    ✅ 91病原体パネル（PMDAと同等）             │
	│    ✅ 検出感度・特異性のバリデーション         │
	└────────────────────────────────────────────────┘

#### 2. Product Testing（製品試験）

	Identity（同一性）:
	- 遺伝子型確認（PCR、シーケンシング）
	- ドナー個体の追跡可能性
	
	Purity（純度）:
	- PERV汚染なし
	- 他の病原体汚染なし
	- 内毒素レベル
	
	Potency（力価）:
	- 臓器機能試験
	- 移植後生存率

#### 3. Preclinical Efficacy Studies（前臨床有効性試験）

	必須:
	✅ 非ヒト霊長類（NHP）研究
	   - 最低3頭のNHP
	   - 最低6ヶ月生存
	   - PERV伝播モニタリング
	
	PERV監視プロトコル（NHP）:
	┌─────────────────────────────────────┐
	│ ベースライン: 移植前血清（PERV陰性確認）│
	│ 移植後:                              │
	│ - Week 1-4: 週次採血                 │
	│ - Month 2-6: 月次採血                │
	│ - Month 6+: 四半期採血               │
	│                                      │
	│ 検査項目:                            │
	│ - qRT-PCR（PERV RNA、血漿）          │
	│ - PCR（PERV DNA、PBMC）              │
	│ - 血清学的検査（抗体）               │
	│ - ウイルス分離培養                   │
	└─────────────────────────────────────┘

### Clinical Trial Requirements（臨床試験）

#### IND（Investigational New Drug）Application

	提出書類:
	1. CMC（Chemistry, Manufacturing, and Controls）
	   - ドナーブタの作製方法
	   - CRISPR/Cas9システム詳細
	   - Off-target解析データ
	   - ロット間一貫性
	
	2. Pharmacology/Toxicology
	   - NHP研究データ
	   - 臓器機能評価
	   - 免疫原性評価
	
	3. Clinical Protocol
	   - 患者選択基準
	   - 手術プロトコル
	   - モニタリング計画
	   - 有害事象対応
	
	4. Investigator's Brochure
	   - PERV-KOブタの科学的背景
	   - 前臨床データサマリー
	   - 既知リスク
	
	5. Informed Consent Form
	   - PERVリスクの説明
	   - 長期モニタリング義務
	   - 密接接触者のリスク

#### 患者モニタリング要件

	頻度:
	┌────────────────────────────────────────┐
	│ Baseline: 移植前（PERV陰性確認必須）   │
	│ Year 1: 1, 3, 6, 12ヶ月                │
	│ Year 2-5: 半年ごと                     │
	│ Year 6+: 年1回、生涯                   │
	│                                        │
	│ 検査項目（各タイムポイント）:         │
	│ ✅ qRT-PCR（PERV RNA、血漿）           │
	│    - 検出限界: <10 copies/mL           │
	│ ✅ PCR（PERV DNA、PBMC）               │
	│ ✅ 血清学（抗体、ELISA/Western）       │
	│    - ≥2抗原陽性 = 感染疑い             │
	│ ✅ アーカイブ検体保存（50年間）        │
	└────────────────────────────────────────┘
	
	陽性判定時のアクション:
	1. 直ちに確認試験（別検体で再検査）
	2. 密接接触者の検査
	3. FDA即時報告（24時間以内）
	4. ウイルス分離・シーケンシング
	5. 公衆衛生当局への報告

#### 密接接触者（Close Contacts）の定義と監視

	定義:
	- 配偶者/パートナー
	- 同居家族
	- 医療従事者（移植チーム）
	
	監視（任意だが推奨）:
	- 年1回のPERVスクリーニング
	- 無症状でも検査継続

## PMDA要件（日本）

### 主要規制文書

1. **厚生労働省「異種移植の実施に伴う公衆衛生上の感染症問題に関する指針」（2016年改訂）**
2. **再生医療等製品の安全性確保に関する法令**
3. **薬機法（医薬品医療機器等法）**

### ドナーブタ要件

#### 1. 飼育管理基準

	必須項目:
	✅ SPF（Specific Pathogen Free）施設での飼育
	✅ 外部病原体侵入防止体制
	✅ 定期的健康診断（3ヶ月ごと）
	✅ 飼育環境記録（温度、湿度、照明）
	✅ 飼料・飲水の管理記録
	✅ 獣医師による定期検査

#### 2. PERV検査要件（PMDAガイドライン準拠）

	┌──────────────────────────────────────────────┐
	│ A. プロウイルスコピー数定量                  │
	│    方法: リアルタイムPCR（ddPCR推奨）        │
	│    標的: PERV pol遺伝子                      │
	│    検出限界: <0.5コピー/ゲノム               │
	│    頻度: 3ヶ月ごと                           │
	│                                              │
	│ B. 感染性ウイルス検出                       │
	│    方法: ブタ末梢血単核球（PBMC）+ ヒト細胞 │
	│    培養: マイトジェン刺激後、共培養          │
	│    期間: 8週間                               │
	│    検出: RT-PCR（週次）、RT活性測定          │
	│    標的細胞: HEK-293またはHT1080             │
	│                                              │
	│ C. PERV-A/B検出                              │
	│    理由: ヒト感染性（PERV-Cは非感染性）      │
	│    方法: サブタイプ特異的PCR                 │
	│    または: Env遺伝子シーケンシング           │
	│                                              │
	│ D. ドナー選択基準                            │
	│    ✅ プロウイルスコピー数が検出限界以下     │
	│    ✅ 感染性ウイルスが検出されない           │
	│    ✅ PERV-A/Bが検出されない                 │
	│    （PERV-KOブタの場合: 全サブタイプ欠損）   │
	└──────────────────────────────────────────────┘

#### 3. 91病原体スクリーニング

	あなたの既存パイプライン（pmda_pathogens.json）が対応:
	✅ 91病原体カバレッジ
	✅ PPA >95%（陽性一致率）
	✅ NPA >98%（陰性一致率）
	
	PERV検証の統合:
	- scripts/phase4_pathogen/perv_typing.py
	- MinION NGSベースの検出
	- 即座SNSアラート（PERV検出時）

### 臨床試験承認プロセス

#### 1. PMDA対面助言（事前相談）

	推奨タイミング: IND申請の6-12ヶ月前
	
	相談内容:
	- PERV検証戦略の妥当性確認
	- 検査方法のバリデーション計画
	- 臨床試験デザイン
	- モニタリング頻度・方法
	
	必要書類:
	- ドナーブタ作製方法詳細
	- CRISPR編集データ（WGS、Off-target解析）
	- PERV検証データ（preliminary）
	- 前臨床研究計画
	
	費用: ¥500,000-2,000,000（相談区分による）
	期間: 申請から面談まで2-3ヶ月

#### 2. 再生医療等製品製造販売承認申請

	提出書類構成:
	┌──────────────────────────────────────┐
	│ 第一部: 申請書・申請資料総括          │
	│                                      │
	│ 第二部: CTD（Common Technical Document）│
	│ - Module 1: 行政情報                 │
	│ - Module 2: CTDサマリー              │
	│ - Module 3: 品質に関する文書         │
	│   → PERV検証データここ               │
	│ - Module 4: 非臨床試験報告           │
	│ - Module 5: 臨床試験報告             │
	│                                      │
	│ 第三部: 参考文献                     │
	└──────────────────────────────────────┘
	
	Module 3（品質）の PERV検証セクション:
	1. ドナー動物の特性
	   - 遺伝的背景（WGSデータ）
	   - CRISPR編集の詳細
	   - Off-target解析結果
	   - 多世代安定性データ
	
	2. PERV検査結果
	   - DNA検査（PCR、ddPCR、WGS）
	   - タンパク質検査（Western Blot）
	   - 機能検査（共培養アッセイ）
	   - 検査方法のバリデーション
	
	3. ロット管理
	   - 世代管理（F0, F1, F2）
	   - 個体識別システム
	   - 検査記録の追跡可能性

#### 3. 条件付き早期承認制度

	日本の特徴: 再生医療の迅速承認
	
	要件:
	✅ 安全性が確認されている
	✅ 有効性が推定される
	✅ 代替治療法がない重篤疾患
	✅ 市販後全例調査の実施計画
	
	利点:
	- 少数例（10-20例）で承認可能
	- 承認後データ蓄積しながら販売
	- 5-7年後に本承認
	
	PERV検証への影響:
	- 条件付き承認でもPERV検証は必須
	- 市販後全例でPERVモニタリング継続
	- 陽性症例発生時の対応計画必須

## 2024年承認事例

### 米国FDA承認例

#### Case 1: NYU Langone Health（2024年1月）

	患者: 腎不全末期
	臓器: 遺伝子編集ブタ腎臓（10遺伝子改変）
	
	PERV検証方法:
	✅ 全ゲノムシーケンス（69遺伝子座のPERV編集確認）
	✅ 共培養アッセイ（8週間、HEK-293細胞）
	✅ RT-PCR（血漿、weekly → monthly）
	⚪ Western Blot（補助データ）
	
	結果: 移植後2ヶ月、PERV伝播なし
	判定: FDA Compassionate Use承認

#### Case 2: University of Maryland（2024年3月）

	患者: 心不全末期
	臓器: 遺伝子編集ブタ心臓（10遺伝子改変）
	
	PERV検証方法:
	✅ WGS（全PERV遺伝子座欠損確認）
	✅ ddPCR（0コピー/ゲノム確認）
	✅ 共培養アッセイ（12週間、multiple cell lines）
	✅ NHP前臨床試験（6ヶ月、PERV陰性）
	
	結果: 移植後6週間生存（心不全死、PERV無関係）
	判定: PERV伝播なし、検証プロトコル有効

### 共通点（2024年承認事例から学ぶ）

	必須検証（両症例共通）:
	1. WGS（全ゲノムレベル確認）
	2. 共培養アッセイ（機能的証明）
	3. NHP前臨床試験（最低6ヶ月）
	4. 患者レシピエント長期モニタリング
	
	任意検証（一部症例のみ）:
	- ddPCR（数値的証明、追加の信頼性）
	- Western Blot（視覚的証明、補助データ）
	- 細胞株パネル（複数ヒト細胞種での確認）
	
	省略可能（両症例で不使用）:
	❌ Flow cytometry
	❌ 免疫組織化学（IHC）
	❌ ELISA（pre-clinical段階）
	❌ 質量分析
	
	結論: Tier 2戦略（WGS + ddPCR + Western + 共培養）が
	      2024年FDA承認事例と完全一致

## クローニング生産の規制要件

### 重要：全頭WGSは規制上不要

**結論**: eGenesisから受領したFounder（創始ブタ）をクローニングで生産する場合、**FDA/PMDA共に全クローン個体のWGSを明示的に要求していない**。

#### FDA/PMDAガイダンスの実態

	FDA "Source Animal, Product, Preclinical, and Clinical Issues" (2016/2024):
	┌──────────────────────────────────────────────────┐
	│ WGS要件の記載:                                    │
	│ ✅ "Genomic characterization of source animals"  │
	│ ✅ "Verification of genetic modifications"       │
	│ ✅ "Stability across generations"                │
	│                                                  │
	│ 明示的に要求されているのは:                      │
	│ - Founderの完全なゲノム特性評価                  │
	│ - 遺伝子編集の確認                              │
	│ - 世代間の安定性確認（代表サンプル）            │
	│                                                  │
	│ 明示的に要求されていないのは:                    │
	│ ❌ "Every cloned animal must undergo WGS"       │
	│ ❌ "100% of production animals require WGS"     │
	│ ❌ "Full genome sequencing for all donors"      │
	└──────────────────────────────────────────────────┘
	
	PMDA「異種移植の実施に伴う公衆衛生上の感染症問題に関する指針」:
	┌──────────────────────────────────────────────────┐
	│ 遺伝的特性評価の記載:                            │
	│ ✅ "ドナー動物の遺伝的背景の明確化"              │
	│ ✅ "遺伝子改変の確認"                            │
	│ ✅ "品質管理のための適切な検査"                  │
	│                                                  │
	│ 明示的に要求されているのは:                      │
	│ - Founderの遺伝的特性評価                        │
	│ - 各個体のPERV検査（PCR/共培養）                 │
	│ - トレーサビリティ（個体識別）                  │
	│                                                  │
	│ 明示的に要求されていないのは:                    │
	│ ❌ 全頭WGS                                       │
	│ ❌ 各クローンの全ゲノム解析                      │
	└──────────────────────────────────────────────────┘

	│ PERV編集部位の変異率: <0.01%                 │
	│                                              │
	│ 根拠となる論文:                              │
	│ 1. Wells et al. (2004) Nature                │
	│    「クローン豚の高い遺伝的忠実度」          │
	│                                              │
	│ 2. Denning et al. (2001) Science             │
	│    「PERV-KO豚の世代間安定性」               │
	│                                              │
	│ 3. Fischer et al. (2020) Nat Biotechnol      │
	│    「10遺伝子編集豚の安定クローニング」      │

	必須検証:
	┌─────────────────────────────────────────────┐

### バリデーション文書

**各検証方法について、以下を文書化**:

	1. 特異性（Specificity）
	   - 他のレトロウイルスと交差反応なし
	   - 陰性コントロール100%陰性
	   - 実証データ: n=20陰性サンプル
	
	2. 感度（Sensitivity）
	   - 検出限界（LOD）の決定
	   - 既知陽性サンプルの検出率
	   - 実証データ: n=10陽性サンプル（段階希釈）
	
	3. 再現性（Reproducibility）
	   - Intra-assay precision (同一日内): CV <10%
	   - Inter-assay precision (日間): CV <15%
	   - 実証データ: 3連 × 3日 = 9測定
	
	4. 直線性（Linearity）
	   - 定量範囲での線形応答
	   - R² >0.99
	   - 実証データ: 6-8ポイント標準曲線
	
	5. 安定性（Stability）
	   - 試薬の有効期限内一貫性
	   - サンプル保存条件最適化
	   - 実証データ: 3ヶ月安定性試験
