## エグゼクティブサマリー

### 外科医からの質問

> 「改変した遺伝子の蛋白が実際に除去・発現しているかの確認をしたほうがいい」

### 回答：3レベルの検証が必要

ゲノム編集されたPERVフリーブタが本当にPERVを持たないことを証明するには、以下の3つのレベルでの検証が必要です：

	レベル1：DNA/遺伝子レベル
	├─ 全ゲノムシーケンス（WGS）
	├─ ddPCR（コピー数定量）
	└─ → PERV遺伝子が存在しないことを確認
	
	レベル2：タンパク質レベル
	├─ Western Blot（推奨）
	├─ ELISA（補助的）
	├─ 免疫組織化学（IHC）
	└─ → PERVタンパク質が発現していないことを確認
	
	レベル3：機能レベル
	├─ 共培養アッセイ（必須）
	├─ RT活性測定
	└─ → 感染性ウイルスが産生されていないことを確認

## 細胞表面タンパク質の発現について

### PERVに感染した細胞の特徴

**主な表面タンパク質**:
- **gp70（エンベロープ表面タンパク質）**: \~54-70 kDa、受容体結合
- **p15E（膜貫通タンパク質）**: \~12-15 kDa、膜融合

**メカニズム**:
	PERV感染
	  ↓
	プロウイルス統合・転写・翻訳
	  ↓
	細胞表面にgp70/p15Eが発現
	  ↓
	新しいウイルス粒子が出芽（budding）
	  ↓
	感染性ウイルス放出

**検出方法**:
- フローサイトメトリー：細胞表面のgp70/p15Eを定量
- 免疫蛍光顕微鏡：組織切片で視覚化
- 免疫組織化学（IHC）：組織内での発現パターン

**PERV-KOブタの場合**:
- 細胞表面にgp70/p15E発現なし = 感染性ウイルス産生なし
- これが「タンパク質レベルでの確認」の意味

## ELISA vs Western Blot

### 結論：併用が最適

| 項目         | ELISA          | Western Blot |
| ---------- | -------------- | ------------ |
| **感度**     | 優秀（5-10 pg/ml） | 良好（ピコグラム範囲）  |
| **特異性**    | 良好（交差反応リスク）    | 優秀（分子量確認）    |
| **スループット** | 高（96-384サンプル）  | 低（10-40サンプル） |
| **時間**     | 1-5時間          | 8-24時間       |
| **コスト**    | 低（¥20-50k）     | 中（¥50-100k）  |
| **定量性**    | 定量的            | 半定量的         |
| **規制受容性**  | ✅ FDA/PMDA承認   | ✅ FDA/PMDA承認 |

### 重要な制約

**ELISAの制約**:
- タンパク質の「存在」を検出するが、「不在」の証明は困難
- 検出限界（LOD）以下と「真の不在」を区別できない
- PERVフリーブタの証明には、複数の方法による多角的検証が必須

**推奨アプローチ**:
1. **Western Blot**: タンパク質不在の一次確認（必須）
2. **ELISA**: 長期臨床モニタリング用（任意）
3. **共培養アッセイ**: 機能的証明（必須）
