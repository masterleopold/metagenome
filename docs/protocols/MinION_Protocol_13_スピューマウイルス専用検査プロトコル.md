# MinION Protocol 13: ブタスピューマウイルス専用検査プロトコル

**バージョン**: 1.0
**作成日**: 2025年11月13日
**対象病原体**: ブタスピューマウイルス（Porcine Spumavirus, SPUMV）
**PMDA分類**: 特別管理微生物 #5
**検査方法**: 条件付きスクリーニング + ネステッドPCR + PERV識別

---- 

## ⚠️ 重要: スピューマウイルス検査の特殊性

### なぜ専用プロトコルが必要か

**スピューマウイルスは他の90病原体と異なる特性を持つ**:

| 項目          | スピューマウイルス            | 他の90病原体           |
| ----------- | -------------------- | ----------------- |
| **参照ゲノム**   | ❌ 存在しない              | ✅ NCBIに存在         |
| **検出方法**    | 交差属検出（30-50%相同性）     | 直接配列比較（\>80%相同性）  |
| **サンプル**    | PBMC ゲノムDNA          | 血漿 cfDNA/cfRNA    |
| **スクリーニング** | 条件付き（5-10%）          | 全サンプル（100%）       |
| **確認試験**    | Sanger配列決定 必須        | NGSのみで完結          |
| **PERV識別**  | 必須（pol遺伝子系統解析）       | 不要                |
| **検出感度**    | 1-10 copies/10⁵ PBMC | 100-500 copies/mL |

**参照ゲノム不在の理由**:
- ブタスピューマウイルスの完全ゲノム配列がNCBIに存在しない（2025年11月時点）
- サル(SFV)、ネコ(FFV)、ウシ(BFV)の配列を代用（Spumaretrovirus属）
- pol遺伝子の保存領域で検出（縮重プライマー使用）

## 📋 科学的背景補足: ブタスピューマウイルスの実在性について

### 重要な事実認識

**本検査プロトコルは、過去70年間で一度も検出されたことのない病原体を対象としています。**

#### 歴史的事実

| 項目           | 事実                    | 備考                 |
| ------------ | --------------------- | ------------------ |
| **検出例**      | ❌ **0件**（1954年〜2025年） | 自然感染、異種移植スクリーニング含む |
| **NCBI登録配列** | ❌ **0件**              | 参照ゲノム、部分配列すべて不在    |
| **論文報告**     | ❌ **0件**              | 査読付き論文での感染報告なし     |
| **抗体検出**     | ❌ **0件**              | 血清学的証拠なし           |
| **ウイルス分離**   | ❌ **0件**              | 細胞培養での分離成功例なし      |

#### なぜPMDA指定91病原体リストに含まれるのか？

**理由1: 予防原則（Precautionary Principle）**
- **系統学的近接性**: ウシ(BFV)、ウマ(EFV)にはスピューマウイルスが存在
- **進化的推論**: ブタも同様にスピューマウイルスを保有する可能性を否定できない
- **理論的リスク**: 存在する場合、持続感染・免疫寛容の特性を持つ

**理由2: 異種間伝播の前例**
- **サルフォーミーウイルス（SFV）**: 霊長類研究者への感染例あり（20-30例）
- **持続感染特性**: 一度感染すると終生持続、無症状キャリア化
- **未知のリスク**: もしブタスピューマウイルスが存在し、ヒトへ伝播した場合の影響不明

**理由3: 検出技術の限界**
- 従来技術では検出できなかった可能性
- NGS・メタゲノム解析の普及により、初めて検出可能に

#### 検出確率の現実的評価

```
【真のスピューマウイルス検出】
確率: <0.1%
根拠: 70年間・数万頭の検査で0件

【PERV偽陽性】
確率: ~5-10%（血漿スクリーニングでトリガー）
      → 系統解析で90%以上がPERVと判明
根拠: pol遺伝子の配列類似性（30-50%）

【結論】
Phase 1でトリガーされた10サンプル中:
  - 9サンプル: PERV（陰性判定）
  - 1サンプル: 判定保留または非特異的増幅
  - 0サンプル: 真のスピューマウイルス（99.9%の確率で）
```

### もし本当に検出されたら何が起きるか

#### シナリオ: 史上初のブタスピューマウイルス検出

**1. 科学的インパクト（研究レベル）**
- **Nature/Science級の論文**: 70年間未検出の病原体の初発見
- **完全長ゲノム配列決定**: MinION long-read + Illumina でカバレッジ500×以上
- **ウイルス分離**: PBMC共培養、電子顕微鏡観察、感染性確認
- **系統進化解析**: Spumaretrovirus属の進化系統樹書き換え

**2. 規制的インパクト（PMDA・異種移植）**
- **ドナーブタ即座除外**: 当該個体は異種移植ドナーとして完全不適格
- **コロニー全体の再評価**: 同一飼育施設の全ブタ再検査
- **PMDA報告**: 特別管理微生物検出として即日報告
- **異種移植プログラム停止**: 感染経路解明まで一時中断の可能性

**3. 国際的インパクト（異種移植業界）**
- **世界初の報告**: 国際異種移植学会（IXA）で緊急発表
- **他国への波及**: 米国・欧州の異種移植プログラムも検査体制見直し
- **ガイドライン改訂**: PMDA・FDA・EMAの指針改訂トリガー

#### 確認試験の追加実施（陽性判定時の必須手順）

もし系統解析でSpumavirus属陽性と判定された場合、**以下の追加確認を必須とする**:

| 確認試験                 | 目的                      | 実施機関      | 期間    |
| -------------------- | ----------------------- | --------- | ----- |
| **1. 独立サンプル再検査**     | 再現性確認                   | 自施設       | 1週間   |
| **2. 別領域PCR増幅**      | pol遺伝子以外（env, gag）の配列確認 | 自施設       | 1週間   |
| **3. MinION完全長配列決定** | 全長ゲノム（\>10 kb）取得        | 自施設       | 2週間   |
| **4. 第三者機関検証**       | 独立した専門機関での確認            | CDC/国立感染研 | 2-4週間 |
| **5. ウイルス分離試験**      | PBMC共培養、電子顕微鏡           | BSL-3施設   | 4-8週間 |
| **6. 血清学的検査**        | 抗Spumavirus抗体検出         | 専門ラボ      | 2週間   |

**判定基準**: 上記6項目中**4項目以上が陽性**の場合のみ、最終的に「ブタスピューマウイルス陽性」と確定

### 本プロトコルを実施する意義

**検出確率が\<0.1%なのに、なぜ検査するのか？**

#### 理由1: 規制遵守（Regulatory Compliance）
- PMDA異種移植指針で**明示的に指定**されている
- 規制当局の要求を満たすことが絶対条件
- 「検出確率が低い」は検査免除の理由にならない

#### 理由2: リスクの大きさ（Risk Severity）
- **確率は低いが、影響は極大**:
  - もし存在すれば、持続感染・ヒト伝播リスク
  - レシピエント（移植患者）の生命に直結
- **後戻りできない**: 一度移植すれば、感染は不可逆

#### 理由3: 科学的誠実性（Scientific Integrity）
- **「不在の証明」は不可能**: 検出されないことが非存在を証明しない
- **技術の進歩**: NGSの登場で初めて検出可能になった可能性
- **未知への備え**: 新興感染症対策の基本原則

#### 理由4: バリデーションデータ蓄積
- **陰性データの価値**: 「N=10,000検査で0件検出」というデータ自体が重要
- **LOD検証**: 検出限界が適切か（偽陰性でないか）の確認
- **科学的貢献**: 「ブタにスピューマウイルスは存在しない」という知見の強化

### 検査実施時の心構え

#### 技術者へのメッセージ

**1. 冷静な判断を**
- Phase 1でトリガーされても、90%以上はPERV
- PCRバンドが出ても、まずPERV誤検出を疑う
- 系統解析で初めて判断する（慌てない）

**2. 厳密な手順遵守を**
- コンタミネーションが最大の敵
- 陽性コントロール（SFV）と陰性コントロールを必ず並行
- PERV識別を確実に実施

**3. もし本当に陽性なら**
- まず再検査で確認（1回の結果で判断しない）
- 上司・プロジェクト責任者に即座に報告
- 第三者機関での検証を依頼
- 歴史的発見の可能性を認識しつつ、慎重に

**4. 陰性判定の価値**
- 「何も出なかった」も重要な結果
- 陰性データがドナー適格性を支持
- 科学的知見への貢献（統計データの一部）

## 3. Phase 2: PBMC採取とゲノムDNA抽出

### 3.1 PBMC分離プロトコル（2時間）

#### 必要材料
- EDTA血液 10 mL（新規採血）
- Ficoll-Paque PLUS 15 mL
- PBS(-) 30 mL
- 50 mL遠心管（滅菌）

#### 手順

**Step 1: Ficoll密度勾配準備（15分）**
1. 50 mL遠心管にFicoll-Paque PLUS 15 mLを分注
2. EDTA血液10 mLをPBS(-) 10 mLで1:1希釈（計20 mL）
3. 希釈血液をFicoll上に**ゆっくり重層**
   4. ピペットを斜めにし、管壁に沿わせて滴下
   5. 2層が混ざらないよう注意

**Step 2: 密度勾配遠心（40分）**
1. 遠心条件: **400 × g, 30分, 室温**
2. **ブレーキなし**（重要: 層が乱れる）
3. 遠心後、4層に分離:
```
┌────────────┐ ← 上層: 血漿（黄色）
├────────────┤ ← PBMC層（白い雲状）← ここを回収
├────────────┤ ← Ficoll層（透明）
└────────────┘ ← 底: 赤血球（赤色）
```

**Step 3: PBMC回収（15分）**
1. パスツールピペットでPBMC層（白濁層）を慎重に吸引
2. 新しい15 mL管に移す（約2-3 mL）
3. PBS(-) 10 mLを加えて希釈

**Step 4: 洗浄（30分）**
1. 遠心: 250 × g, 10分, 室温
2. 上清を捨て、PBS(-) 10 mLで再懸濁
3. **2回洗浄を繰り返す**

**Step 5: 細胞数カウント（10分）**
1. トリパンブルー染色（1:1希釈）
2. 血球計算盤でカウント
3. **期待収量**: 5-15 × 10⁶ cells（10⁷ cellsあれば理想的）

**Step 6: 保存**
- 即座にDNA抽出に進む（推奨）
- または-80°Cで凍結保存（1ヶ月以内）

### 3.2 ゲノムDNA抽出（2時間）

#### QIAamp DNA Blood Mini Kitプロトコル

**Step 1: 細胞溶解（15分）**
1. PBMC 5 × 10⁶ cells（約1 mL）を1.5 mL tubeに移す
2. PBS(-) でトータル200 μLに調整
3. Proteinase K 20 μL添加
4. Buffer AL 200 μL添加、ボルテックス15秒
5. 56°C、10分インキュベート

**Step 2: エタノール沈殿（5分）**
1. エタノール（99.5%）200 μL添加
2. ボルテックス15秒

**Step 3: カラム精製（30分）**
1. 混合液全量をQIAamp Miniカラムに移す
2. 遠心: 8,000 rpm, 1分
3. フロースルーを廃棄
4. Buffer AW1 500 μL添加、遠心: 8,000 rpm, 1分
5. Buffer AW2 500 μL添加、遠心: 14,000 rpm, 3分
6. 空遠心: 14,000 rpm, 1分（完全乾燥）

**Step 4: DNA溶出（10分）**
1. 新しい1.5 mL tubeにカラムをセット
2. Buffer AE（またはTE buffer）100 μL添加
3. 室温、1分静置
4. 遠心: 8,000 rpm, 1分

**Step 5: 定量とQC（20分）**
1. Qubit dsDNA HS Assay で定量
   2. **期待濃度**: 20-100 ng/μL
   3. **総収量**: 2-10 μg
2. NanoDrop で純度確認
   5. **260/280**: 1.8-2.0
   6. **260/230**: \>2.0
3. 必要に応じて1% アガロースゲルで確認
   8. 高分子量DNA（\>10 kb）を確認

## 4. Phase 3: ネステッドPCR（pol遺伝子増幅）

### 4.1 縮重プライマー設計

**pol遺伝子の保存領域をターゲット**

#### Outer Primers（1st PCR）

| プライマー            | 配列（5'→3'）            | 縮重度     | Tm   | 備考         |
| ---------------- | -------------------- | ------- | ---- | ---------- |
| **SPUMV-pol-F1** | GGNCARATHGGNATGTTYGG | 96-fold | 55°C | pol遺伝子 N末端 |
| **SPUMV-pol-R1** | CCRTCNCCRAANCCRTC    | 64-fold | 53°C | pol遺伝子 中央部 |

**期待増幅産物**: \~800 bp

#### Inner Primers（2nd PCR）

| プライマー            | 配列（5'→3'）            | 縮重度     | Tm   | 備考      |
| ---------------- | -------------------- | ------- | ---- | ------- |
| **SPUMV-pol-F2** | ATHGGNCARGGNTTYACNAC | 96-fold | 56°C | 1st産物内側 |
| **SPUMV-pol-R2** | GTRTCNGTYTTRTCNCC    | 64-fold | 52°C | 1st産物内側 |

**期待増幅産物**: \~400 bp

**縮重塩基の説明**:
- N = A/T/G/C（4通り）
- R = A/G（2通り）
- Y = C/T（2通り）
- H = A/C/T（3通り）

### 4.2 1st PCR（Outer PCR）プロトコル（3時間）

#### 反応液組成（50 μL）

| 成分                           | 濃度          | 体積 (μL)  |
| ---------------------------- | ----------- | -------- |
| **テンプレートDNA**                | 100 ng      | 2.0      |
| 10× High Fidelity PCR Buffer | 1×          | 5.0      |
| 50 mM MgSO₄                  | 2 mM        | 2.0      |
| 10 mM dNTP mix               | 0.2 mM each | 1.0      |
| **SPUMV-pol-F1** (10 μM)     | 0.5 μM      | 2.5      |
| **SPUMV-pol-R1** (10 μM)     | 0.5 μM      | 2.5      |
| Platinum Taq High Fidelity   | 1 U         | 0.2      |
| 滅菌水                          | -           | 34.8     |
| **合計**                       |             | **50.0** |

#### サーマルサイクル条件

```
初期変性:  94°C  3分     × 1サイクル
────────────────────────────────
変性:      94°C  30秒 ┐
アニーリング: 50°C  30秒 │ × 35サイクル
伸長:      72°C  1分  ┘
────────────────────────────────
最終伸長:  72°C  7分     × 1サイクル
保持:       4°C  ∞
```

**重要ポイント**:
- **低ストリンジェンシー**: 縮重プライマーのため、アニーリング温度50°Cと低め
- **サイクル数多め**: 35サイクル（通常30サイクルより多い）
- **長い伸長時間**: 1分/kb（800 bpなので1分）

#### 電気泳動確認（30分）

1. 1.5% アガロースゲル作成（TBE buffer）
2. PCR産物 5 μL + Loading Dye 1 μL をロード
3. 100V、30分泳動
4. EtBr染色（10分）または SafeView染色
5. **期待バンド**: \~800 bp
   6. バンドが確認できれば2nd PCRへ
   7. バンドなし → トラブルシューティングへ

### 4.3 2nd PCR（Inner PCR）プロトコル（3時間）

#### 1st PCR産物の希釈

**重要**: 1st PCR産物を直接使用すると阻害が起きる

1. 1st PCR産物を滅菌水で **1:100希釈**
   2. 例: 1 μL 1st産物 + 99 μL 滅菌水
2. この希釈液をテンプレートとして使用

#### 反応液組成（50 μL）

| 成分                           | 濃度          | 体積 (μL)  |
| ---------------------------- | ----------- | -------- |
| **1st PCR産物（1:100希釈）**       | -           | 1.0      |
| 10× High Fidelity PCR Buffer | 1×          | 5.0      |
| 50 mM MgSO₄                  | 2 mM        | 2.0      |
| 10 mM dNTP mix               | 0.2 mM each | 1.0      |
| **SPUMV-pol-F2** (10 μM)     | 0.5 μM      | 2.5      |
| **SPUMV-pol-R2** (10 μM)     | 0.5 μM      | 2.5      |
| Platinum Taq High Fidelity   | 1 U         | 0.2      |
| 滅菌水                          | -           | 35.8     |
| **合計**                       |             | **50.0** |

#### サーマルサイクル条件

```
初期変性:  94°C  3分     × 1サイクル
────────────────────────────────
変性:      94°C  30秒 ┐
アニーリング: 52°C  30秒 │ × 30サイクル
伸長:      72°C  45秒 ┘
────────────────────────────────
最終伸長:  72°C  7分     × 1サイクル
保持:       4°C  ∞
```

**1st PCRとの違い**:
- アニーリング温度: 50°C → **52°C**（Inner primerの方が特異性高い）
- サイクル数: 35 → **30**（テンプレートがすでに増幅済み）
- 伸長時間: 1分 → **45秒**（400 bpと短い）

#### 電気泳動確認（30分）

1. 1.5% アガロースゲル作成
2. 2nd PCR産物 5 μL + Loading Dye 1 μL をロード
3. 100V、30分泳動
4. **期待バンド**: \~400 bp（鮮明なバンド1本）

**判定**:
- ✅ **400 bp 単一バンド** → Phase 4（精製・Sanger）へ
- ❌ **バンドなし** → 陰性（PERVの可能性、またはスピューマウイルス不在）
- ⚠️ **複数バンド** → 非特異的増幅、プライマー再設計必要

### 4.4 PCRコントロール

**陽性コントロール（必須）**:
- Simian foamy virus (SFV) ゲノムDNA
- 入手先: ATCC VR-56 または類似株
- 期待結果: 両PCRで明確なバンド

**陰性コントロール（必須）**:
- 滅菌水（テンプレートなし）
- 期待結果: バンドなし

**PERV識別コントロール（推奨）**:
- PERV-A/B/C プラスミド（pol遺伝子クローン）
- 期待結果: 系統解析でPERVクラスターに分類される

## 5. Phase 4: Sanger配列決定

### 5.1 PCR産物精製（1時間）

#### QIAquick Gel Extraction Kitプロトコル

**Step 1: ゲル切り出し（15分）**
1. UVトランスイルミネーター上で400 bpバンドを確認
2. クリーンなカッターで**目的バンドのみ**を切り出し
3. 1.5 mL tubeに移し、重量測定
   4. **期待重量**: 50-100 mg

**Step 2: ゲル溶解（10分）**
1. Buffer QG（ゲル重量の3倍量）添加
   2. 例: 50 mg ゲル → 150 μL Buffer QG
2. 50°C、10分インキュベート（ボルテックス時々）
3. ゲルが完全に溶解することを確認（黄色透明）

4. Buffer PE 750 μL で洗浄（2回）

1. Buffer EB 30 μL をカラム中央に添加

4. **溶出液（精製PCR産物）を回収**

**Step 5: 定量（10分）**
1. NanoDrop で濃度測定
   2. **期待濃度**: 10-50 ng/μL

2. または Qubit dsDNA HS Assay

## 8. トラブルシューティング

### 8.1 PBMC分離の問題

#### 問題1: PBMC層が見えない

**原因**:
- Ficoll層と血液層の混合
- ブレーキ付き遠心
- 血液量不足

**対策**:
1. **ブレーキなし**遠心を確認
2. Ficoll上に血液を**ゆっくり重層**（ピペットを斜めに）
3. 血液量を10 mL以上に増やす

#### 問題2: PBMC収量が少ない（\<5 × 10⁶）

**原因**:
- 血液の凝固
- PBMC層の回収ミス
- ブタの白血球数が低い

**対策**:
1. 採血後**4時間以内**に処理
2. PBMC層を広めに回収（Ficoll層上部も含める）
3. 必要に応じて血液量を20 mLに増やす

### 8.2 PCRの問題

#### 問題3: 1st PCRでバンドが出ない

**原因**:
- テンプレートDNA量不足
- プライマーのミスマッチ
- PCR阻害物質

**対策**:
1. テンプレートDNA量を200 ngに増量
2. アニーリング温度を**48°C**に下げる（グラデーションPCR推奨）
3. BSA（100 μg/mL）を添加してPCR阻害を軽減
4. 新しいDNA抽出を試す

#### 問題4: 2nd PCRでバンドが出ない

**原因**:
- 1st PCR産物の希釈不足
- PCR阻害
- テンプレート添加忘れ

**対策**:
1. 1st PCR産物を**1:10希釈**に変更（濃くする）
2. 1st PCRゲル精製後に2nd PCRを実施
3. 陽性コントロール（SFV DNA）で系全体を確認

#### 問題5: 複数バンドが出現（非特異的増幅）

**原因**:
- アニーリング温度が低すぎる
- プライマー濃度が高すぎる
- サイクル数が多すぎる

**対策**:
1. アニーリング温度を**54°C**に上げる
2. プライマー濃度を0.3 μMに下げる
3. サイクル数を25サイクルに減らす
4. Touchdown PCRを試す:
```
94°C 30秒 ┐
58°C 30秒 │ × 10サイクル（-0.5°C/サイクル）
72°C 45秒 ┘
94°C 30秒 ┐
52°C 30秒 │ × 20サイクル
72°C 45秒 ┘
```

### 8.3 Sanger配列決定の問題

#### 問題6: シーケンス品質が低い（Q20 \<80%）

**原因**:
- PCR産物の精製不十分
- プライマー残存
- テンプレート濃度不適

**対策**:
1. QIAquick精製を**2回**繰り返す
2. エタノール沈殿で再精製
3. テンプレート濃度を調整（5-10 ng/μL）

#### 問題7: 配列が読めない（両方向不一致）

**原因**:
- 複数バンドの混入
- プライマーダイマー
- 非特異的増幅産物

**対策**:
1. ゲル精製を**より厳密に**（目的バンドのみ切り出し）
2. 別のクローニング後に配列決定
3. MinION direct sequencingを試す（代替法）

### 8.4 系統解析の問題

#### 問題8: Bootstrap値が低い（\<50%）

**原因**:
- アライメント長が短い（\<200 bp）
- 配列品質が低い
- 参照配列の選択不適切

**対策**:
1. より長いPCR産物を増幅（600-800 bp）
2. 配列決定の再実施
3. 追加の参照配列を含める（他のSpumavirusを追加）
4. Bootstrap回数を2000に増やす

#### 問題9: クラスターが不明瞭

**原因**:
- サンプルがどのクレードにも属さない
- 新規スピューマウイルス株の可能性
- 配列エラー

**対策**:
1. BLAST検索でトップヒットを確認
2. 再度PCR → Sanger配列決定
3. PBMC共培養を試す（ウイルス分離）
4. MinION long-read sequencingで全長配列取得

## 付録A: プライマー配列詳細

### 縮重プライマーの設計根拠

#### pol遺伝子の保存領域解析

スピューマウイルスpol遺伝子は以下の保存ドメインを持つ:
1. **プロテアーゼ（PR）ドメイン**
2. **逆転写酵素（RT）ドメイン** ← ターゲット
3. **インテグラーゼ（IN）ドメイン**

本プロトコルでは**RTドメイン**の保存領域をターゲットとする。

#### 配列アライメント（参照）

```
SFV   : GGTCAAATGGGTATGTTTGG
FFV   : GGCCAGATTGGGATGTACGG
BFV   : GGGCAAATTGGGCTGTATGG
Consensus: GGNCARATHGGNATGTTYGG  ← Outer Forward

SFV   : CCATCGCCAAAGCCATC
FFV   : CCGTCTCCGAACCCCTC
BFV   : CCTTCACCCAACCCTTC
Consensus: CCRTCNCCRAANCCRTC  ← Outer Reverse
```

### プライマー入手方法

**推奨サプライヤー**: Sigma-Aldrich、IDT（Integrated DNA Technologies）

**発注時の指定**:
- 精製レベル: **HPLC精製**（必須、縮重プライマーのため）
- スケール: 50 nmol（十分な量）
- 濃度: 100 μM（水溶液）

**コスト**: 約¥8,000（4種類合計）
