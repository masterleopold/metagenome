## トレーニング目標

本トレーニング終了時、参加者は以下ができるようになります：

1. **説明**: 環状DNAおよび1本鎖DNAウイルスが特別な処理を必要とする理由を説明できる
2. **同定**: Step 2.5を必要とする4種のPMDA病原体（PCV2、PCV3、TTV、PPV）を同定できる
3. **実施**: DNase I超低濃度直鎖化処理を独立して実施できる
4. **実行**: Klenow Fragment 2本鎖合成を95%以上の成功率で実行できる
5. **トラブルシューティング**: 一般的な問題（直鎖化失敗、合成不完全、過消化）を解決できる
6. **解釈**: QC結果（Bioanalyzerトレース、Qubit定量）を解釈できる
7. **記録**: ALCOA+原則に準拠してStep 2.5を記録できる

#### 09:00-10:00 | モジュール1：理論的背景（1時間）
**学習目標**: Step 2.5が必要な理由を理解する

**カバートピック**:
1. **PMDA 91病原体要件**
   2. 厚労省別添2指定病原体リスト
   3. PPA \>95%要件（全病原体）
   4. 特別管理病原体（PCV2、PCV3）

2. **DNAゲノム構造の多様性**
   2. 直鎖dsDNA（大部分のDNAウイルス）
   3. 環状dsDNA（アデノウイルス、SV40）
   4. 環状ssDNA（サーコウイルス、TTV）
   5. 直鎖ssDNA（パルボウイルス）

3. **SQK-LSK114ライゲーションキットの制限**
   2. NEBNext Ultra II末端修復は遊離3'/5'末端を必要とする
   3. T4 DNAリガーゼ: dsDNAで\>95%効率、ssDNAで\<5%効率
   4. MinIONモータータンパク質はdsDNA入力を必要とする

4. **環状DNAが直鎖化なしでシーケンス不可能な理由**
	環状DNA（例：PCV2 1.7 kb）:
	
	     ╔════════════════╗
	     ║  遊離末端なし  ║  ← アダプター結合不可
	     ╚════════════════╝
	
	DNase I直鎖化後:
	
	     5'═══════════════3'
	     3'═══════════════5'  ← 遊離末端利用可能

5. **ssDNAが2本鎖合成なしでシーケンス不可能な理由**
	1本鎖DNA（例：PPV 5.0 kb）:
	
	     5'─────────────────3'  ← T4リガーゼ <5%効率
	
	Klenow Fragment合成後:
	
	     5'═══════════════3'
	     3'═══════════════5'  ← T4リガーゼ >95%効率

**評価**: 5問クイズ（合格スコア：4/5）

#### 10:00-11:00 | モジュール2：Step 2.5プロトコル詳細（1時間）
**学習目標**: 各サブステップを詳細に理解する

**Sub-step 2.5.1: 環状DNA直鎖化（合計30分）**

**原理**: DNase Iが環状DNAにランダムな一本鎖ニックを作り、閉環状から直鎖分子に変換する。

**重要パラメータ**:
- **DNase I濃度**: 0.005 U（超低濃度、標準PCRクリーンアップの1/2000）
- **温度**: 37°C（最適酵素活性）
- **時間**: 5分（限定消化、完全分解ではない）
- **失活**: 75°C、10分（不可逆的熱変性）

**ステップバイステップ**:
1. 直鎖化マスターミックス調製（氷上）:
   2. DNAサンプル（CpG除去後）: 20 μL
   3. DNase I Buffer（10×）: 3 μL
   4. ヌクレアーゼフリー水: 6.5 μL
   5. DNase I（0.005 U）: 0.5 μL
   6. **総容量**: 30 μL

2. 気泡なしで穏やかに混合（5回ピペッティング）

3. インキュベート 37°C、5分（PCR機またはヒートブロック）

4. 熱失活 75°C、10分

5. 室温まで冷却（ベンチ上で2分）

6. AMPure XP精製（0.8×比率）:
   2. 24 μL ビーズ追加（30 μLの0.8×）
   3. ピペッティングで10回混合
   4. RT、5分インキュベート
   5. マグネット 5分（透明になるまで）
   6. 上清除去（トラブルシューティング用に保存）
   7. 200 μL 80% EtOHで2回洗浄
   8. 5分風乾（マット仕上げまで、ひび割れなし）
   9. 21 μL ヌクレアーゼフリー水で溶出
   10. 最終容量: 20 μL 直鎖化DNA

**期待結果**:
- Qubit: 70-90%回収（入力が10 ngの場合、7-9 ngを期待）
- Bioanalyzer: わずかなサイズシフト（環状→直鎖移動度変化）

---- 

**Sub-step 2.5.2: ssDNA → dsDNA変換（合計2時間）**

**原理**: ランダムヘキサマープライマーがssDNA鋳型に結合し、Klenow Fragmentが相補鎖を合成する。

**重要パラメータ**:
- **Klenow Fragment（exo-）**: 3'→5'エクソヌクレアーゼ欠損（鋳型分解を防ぐ）
- **ランダムヘキサマー**: 50 μMストック、5 μM最終濃度（飽和濃度）
- **変性**: 95°C、3分（2次構造を変性）
- **アニーリング**: 25°C、5分（ヘキサマー結合）
- **伸長**: 16°C、60分（ゆっくり、プロセシビティ〜1 kb/分）

**ステップバイステップ**:

**フェーズ1: 変性（5分）**
1. 直鎖化DNA 20 μLをPCRチューブに移す
2. 変性: 95°C、3分（PCR機）
3. 直ちに氷に移す（スナップクール、2分）
   4. **重要**: 相補領域の再アニーリングを防ぐ

**フェーズ2: アニーリング（10分）**
4. アニーリングマスターミックス調製（氷上）:
   2. 変性DNA: 20 μL
   3. ランダムヘキサマー（50 μM）: 2 μL
   4. **総量**: 22 μL

5. 穏やかに混合、スピンダウン

6. アニール: 25°C、5分（ヒートブロックまたはPCR機）
   2. **結果**: ヘキサマーがssDNA鋳型に100-200 bp毎に結合

**フェーズ3: 伸長（90分）**
7. 伸長マスターミックス調製（RTで保持）:
   2. アニールDNA + ヘキサマー: 22 μL
   3. NEBNext Ultra II Reaction Buffer（10×）: 3 μL
   4. dNTPミックス（各10 mM）: 3 μL
   5. ヌクレアーゼフリー水: 1.5 μL
   6. Klenow Fragment（exo-、5 U/μL）: 0.5 μL
   7. **総量**: 30 μL

8. 気泡なしで穏やかに混合（5回ピペッティング）

9. インキュベート 16°C、60分（PCR機）
   2. **なぜ16°C？** プライマー安定性（より高いTm）とKlenow活性のバランス

10. 熱失活 75°C、10分

11. 室温まで冷却（ベンチ上で2分）

**フェーズ4: 精製（30分）**
12. AMPure XP精製（1.8×比率）:
	- 54 μL ビーズ追加（30 μLの1.8×）
	- ピペッティングで10回混合
	- RT、5分インキュベート
	- マグネット 5分（透明になるまで）
	- 上清除去（トラブルシューティング用に保存）
	- 200 μL 80% EtOHで2回洗浄
	- 5分風乾（マット仕上げまで）
	- 21 μL ヌクレアーゼフリー水で溶出
	- 最終容量: 20 μL dsDNA

**期待結果**:
- Qubit: 80-95%回収（入力が8 ngの場合、6.4-7.6 ngを期待）
- Bioanalyzer: ピークサイズ不変（サイズは同じだが、現在dsDNA）
- **重要指標**: Bioanalyzer濃度 ≈ Qubit の2倍（dsDNAは色素をより多く取り込む）

#### 11:00-12:00 | モジュール3：品質管理とトラブルシューティング（1時間）
**学習目標**: QC結果を解釈し、失敗をトラブルシューティングする

**QCチェックポイント**:

**チェックポイント1: Sub-step 2.5.1後（直鎖化）**
- **Qubit dsDNA HS**: 70-90%回収
  - ✅ 合格: 7-9 ng（入力が10 ngの場合）
  - ⚠️ 警告: 5-7 ng（限界回収、続行可）
  - ❌ 不合格: \<5 ng（50%損失、続行前にトラブルシューティング）

- **Bioanalyzer DNA HS**:
  - ✅ 合格: 単一ピーク、スメアなし、予想サイズ
  - ❌ 不合格: スメアまたは複数ピーク（過消化）

**チェックポイント2: Sub-step 2.5.2後（2本鎖合成）**
- **Qubit dsDNA HS**: 80-95%回収
  - ✅ 合格: 6.4-7.6 ng（入力が8 ngの場合）
  - ❌ 不合格: \<5 ng（合成やり直し）

- **Bioanalyzer DNA HS**:
  - ✅ 合格: Bioanalyzer濃度 ≈ Qubitの2倍（dsDNA特性）
  - ❌ 不合格: Bioanalyzer = Qubit（まだssDNA、合成不完全）

---- 

**一般的な問題と解決策**:

| 問題                         | 原因                | 解決策                             |
| -------------------------- | ----------------- | ------------------------------- |
| **直鎖化後\<50%回収**            | DNase I過消化        | DNase Iを0.003 Uに減らすか時間を3分に      |
| **Bioanalyzerでスメア**        | DNase I過消化        | 新鮮なDNase Iを使用、濃度を減らす            |
| **直鎖化なし（環状DNA残存）**         | DNase I不活性        | DNase I保存(-20°C)を確認、新鮮なアリコートを使用 |
| **合成後Bioanalyzer = Qubit** | 2本鎖合成不完全          | Klenowを1 Uに増やすか90分に延長           |
| **合成後\<50%回収**             | Klenowエクソヌクレアーゼ汚染 | Klenow（exo-）のみ使用、通常Klenowは不可    |
| **合成後産物なし**                | ランダムヘキサマー分解       | 新鮮なランダムヘキサマーを使用、-20°C保存         |

	├─ アニール済みミックス: 22 μL
	├─ Reaction Buffer（10×）: 3 μL
	├─ dNTPミックス（10 mM）: 3 μL
	├─ ヌクレアーゼフリー H₂O: 1.5 μL
	└─ Klenow（exo-）: 0.5 μL
	→ 16°C、60分
	→ 75°C、10分（失活）
	
	精製（AMPure XP 1.8×）:
	1. 54 μL ビーズ追加、10回混合
	2. RT 5分、マグネット 5分
	3. 上清除去（保存！）
	4. 200 μL 80% EtOHで2回洗浄
	5. 5分風乾（マット仕上げ）
	6. 21 μL H₂Oで溶出、20 μL回収
	
	QCチェックポイント:
	✓ Qubit: 80-95%回収
	✓ Bioanalyzer ≈ Qubitの2倍（dsDNA指標）
	
	出力: 20 μL dsDNA → DNAライブラリー調製（Step 3）
