## サンプル調製

### 参照標準品調製

**ステップ1: ストック溶液調製**（-30日目）

1. **合成DNA構築物を再懸濁**:
   2. PCV2環状ssDNA（5 μg凍結乾燥）
   3. PCV3環状ssDNA（5 μg凍結乾燥）
   4. TTV環状ssDNA（5 μg凍結乾燥）
   5. PPV直鎖ssDNA（5 μg凍結乾燥）

2. **初期定量**（三連測定）:
   2. Qubit dsDNA HSアッセイ
   3. Bioanalyzer DNA HSチップ
   4. 分子量計算: MW = （サイズ bp）× 650 Da/bp

3. **コピー数計算**:
```
copies/μL = (ng/μL × 6.022×10²³) / (MW Da × 10⁹)

例：PCV2（1767 bp、10 ng/μL）:
MW = 1767 × 650 = 1,148,550 Da
copies/μL = (10 × 6.022×10²³) / (1.14855×10⁶ × 10⁹)
           = 5.24×10⁹ copies/μL
```

4. **作業用ストック調製**:
   2. TEバッファー（10 mM Tris、1 mM EDTA、pH 8.0）で1×10⁷ copies/μLに希釈
   3. 50 μL使い切りチューブにアリコート
   4. -80°Cで保存（2年間安定）
   5. 凍結融解サイクル回避（最大2サイクル）

**ステップ2: 血漿マトリックススパイキング**（各実験日）

## 統計解析

### LOD計算方法

#### 方法1: プロビット解析（推奨）

**原理**: プロビット回帰を用いて濃度の関数として検出確率をモデル化

**ソフトウェア**: R（パッケージ: `MASS`）、GraphPad Prism、またはSPSS

**ステップ**:
1. 検出をコード化: 1 = 検出、0 = 非検出
2. プロビットモデル適合: Φ⁻¹(P) = β₀ + β₁ × log₁₀(濃度)
3. LOD95計算: 95%検出確率の濃度
4. 尤度比検定を用いて95%信頼区間計算

**出力**:
- LOD95（copies/mL）± 95% CI
- プロビットプロット（S字曲線）

**Rコード例**:
```R
library(MASS)

# データ: 濃度（copies/mL）、検出（1/0）
data <- data.frame(
  concentration = rep(c(50, 100, 200, 300, 500, 1000, 1500, 2000), each=20),
  detected = c(...)  # 二値検出結果
)

# プロビットモデル適合
model <- glm(detected ~ log10(concentration),
             family=binomial(link="probit"), data=data)

# LOD95計算
LOD95 <- 10^((qnorm(0.95) - coef(model)[1]) / coef(model)[2])
print(paste("LOD95 =", round(LOD95, 0), "copies/mL"))

# confint()を用いた95% CI
ci <- confint(model)
```

---- 

#### 方法2: ヒット率解析（代替）

**原理**: 各濃度での経験的ヒット率

**ステップ**:
1. ヒット率計算: （検出数）/（総数）各濃度で
2. ヒット率≥95%の最低濃度特定
3. Clopper-Pearson厳密法を用いて95% CI計算

**例**:
```
濃度: 500 copies/mL
反復: 20
検出: 19
ヒット率: 19/20 = 95%
95% CI: [75.1%, 99.9%]（Clopper-Pearson）
```

### 再現性解析

#### 日間CV%計算

**式**:
```
CV% = (SD / 平均) × 100%

ここで:
SD = √[Σ(xi - x̄)² / (n-1)]
x̄ = 日間平均ヒット率
n = 日数（10）
```
