**作成日**: 2025年10月8日

### 臨床試験フェーズの特性

```yaml
Clinical_Trial_Phase_Characteristics:
  移植頭数: "2-6頭/年（初期臨床試験）"

  サンプリング戦略:
    育成期間中の定期モニタリング:
      - 頻度: "月1回 × 3-6ヶ月"
      - サンプル数: "3-6サンプル/頭"

    移植直前:
      - 最終スクリーニング: "1サンプル/頭"
      - 追加確認: "1サンプル/頭"
      - 小計: "2サンプル/頭"

    1頭あたり総サンプル数: "5-8サンプル"

  年間総サンプル数:
    Scenario_Low: "2頭 × 5サンプル = 10サンプル/年"
    Scenario_Medium: "4頭 × 6サンプル = 24サンプル/年"
    Scenario_High: "6頭 × 8サンプル = 48サンプル/年"

  バッチ解析戦略:
    概念: "1頭分の全サンプル（5-8個）を移植直前に一括解析"
    利点:
      - "装置稼働回数削減"
      - "試薬ロス最小化"
      - "時系列比較が容易"
      - "コスト最適化"

  重要な制約:
    - "移植スケジュールは不定期（ブタ成長次第）"
    - "緊急時の追加検査需要あり"
    - "長期サンプル保存が必要（-80°C, 3-6ヶ月）"
    - "装置稼働率低い（年間4-8回のみ）"
```


## 7. 結論とアクションアイテム

### 7.1 明確な結論

```
╔════════════════════════════════════════════════════════════╗
║  臨床試験フェーズ NGSプラットフォーム 最終推奨              ║
╚════════════════════════════════════════════════════════════╝

【推奨装置】 Illumina MiSeq System with v3 (600-cycle) kit

【推奨理由】
1. 初期投資: 15M円（最小、NextSeqの60%）
2. 柔軟性: 1-96サンプル対応（臨床試験の変動に最適）
3. 操作性: 簡単、兼任スタッフで運用可能
4. 性能: LOD 50-100 copies/mL（必要十分）
5. コスト: 5年間で2,400万円節約（vs NextSeq）

【サンプルあたりコスト】
- 変動費: 53,000-66,000円/サンプル
- 総コスト: 147,000-244,000円/サンプル（検体数により変動）

【バッチ戦略】
- Phase I (10サンプル/年): 1頭ずつ解析
- Phase II (24サンプル/年): 2頭まとめ（12サンプル/run × 2回）
- Phase III (48サンプル/年): 3頭まとめ（24サンプル/run × 2回）

【NextSeq 2000との比較】
                    MiSeq      NextSeq 2000    差額
初期投資           15M円      25M円          -10M円
5年総コスト        49.8M円    73.8M円        -24M円
サンプルコスト     220K円     327K円         -107K円

結論: MiSeqがあらゆる面で優位（臨床試験フェーズ）

【NextSeq検討タイミング】
- 年間サンプル数 >100
- 外部受託サービス開始時
- Phase III終了後の実用化フェーズ移行時
```
